:Overview: Performs the demultiplexing of your raw Illumina data
:Input: bcl files from an Illumina sequencer
:Output: fastq files

Usage
~~~~~~~

::

    sequana_pipelines_demultiplex --input-directory

~~~~~~~~~~~~~~~~~~

This pipeline is to be used with bcl2fastq 2.20.0
